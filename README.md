vagrant-nexus
=============

Project to set up Sonatype Nexus via Vagrant / Puppet.

Quick start
-----------

* `git clone git@github.com:skempken/vagrant-nexus.git`
* `cd vagrant-nexus`
* `git submodule init`
* `git submodule update`
* `vagrant up`
* ...wait...
* Nexus should be available on `http://localhost:9000`
